# Property Planner

A property scenario planner built with React + Vite.

## Stack

- React 19
- Vite
- Recharts

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `claude/setup-react-vite-project-mYjRT` automatically deploy to GitHub Pages via the Actions workflow.

Live site: https://staceyjf.github.io/Property-planner/
